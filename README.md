# dnbradio-player

> dnbradio-player

## Build Setup

``` bash
# install dependencies
$ pnpm i

# serve with hot reload on your localhost
$ pnpm run dev

# build for production and launch server
$ pnpm run build
$ pnpm run start

# generate static project
This is what is currently deployed to the dnbradio.com website.
$ pnpm run generate
```
